## 1. 设计要求的验证项（6.2.2 + 6.2.3）

每个用例需验证三方面（6.2.3）：

1. **返回值**：Update 返回的 Model 数据正确。
2. **关系表**：通过 Query 验证关系表中的链接数据与预期一致。
3. **关联实体表**：通过 Query 验证引用关系下被解除引用的实体仍存在于库中（不被删除）。
